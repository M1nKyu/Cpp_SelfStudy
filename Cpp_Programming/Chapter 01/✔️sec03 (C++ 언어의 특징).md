# 제네릭 프로그래밍
- 제네릭 함수
	- 동일한 프로그램 코드에서 다양한 데이터 타입을 적용할 수 있게 일반화 시킨 함
- 제네릭 클래스
	- 동일한 프로그램 코드에 다양한 데이터 타입을 적용할 수 있게 일반화 시킨 클래스
- template 키워드로 선언
	- 템플릿 함수 / 템플릿 클래스 라고도 부름
- 제네릭 프로그래밍
- 제네릭 함수와 제네릭 클래스를 활용하여 프로그램을 작성하는 새로운 패러다임
